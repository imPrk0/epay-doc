# 订单状态枚举 {#title}

订单状态为 Integer 数字类型，具体值和含义如下：

| 状态 | 说明 |
| :---: | :--- |
| `0` | 未支付 |
| `1` | 已支付 |
| `2` | 已退款 |
| `3` | 已冻结 |
| `4` | 预授权 |
