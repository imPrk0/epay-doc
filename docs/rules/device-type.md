# 设备类型枚举 {#title}

设备类型为 String 字符串类型，具体值和含义如下：

| 状态 | 说明 |
| :---: | :--- |
| `pc` | 电脑浏览器 _**<font color="red">默认</font>**_ |
| `mobile` | 手机浏览器 |
| `qq` | 手机 QQ 内浏览器 |
| `wechat` | 微信内浏览器 |
| `alipay` | 支付宝客户端 |
| `douyin` | 抖音APP |
