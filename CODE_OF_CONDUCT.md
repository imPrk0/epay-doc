# 代码贡献者行为准则

**本准则旨在确保所有项目的贡献者在一个安全、包容且专业的环境中工作。无论职位高低，每位成员都有责任遵守并维护这些标准。**


## 价值观

- **平等与尊重**<br />
  每个人都应该有机会参与开源项目，不论年龄、性别、种族与宗教等。我们鼓励多样性，并致力于创造一个包容的社区环境。无论出身，我们生来平等，我们应当彼此尊重。
- **人权至上**<br />
  我们承认并尊重每个人的基本人权。我们反对任何形式的歧视、骚扰和不公平待遇，并致力于创造一个安全和包容的环境。**如果法律与人权发生冲突时，我们应当尽我们所能维护人权的最高标准！**
